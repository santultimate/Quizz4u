1. Introduction et Engagement

Quizz4U ("nous", "notre application", "le développeur") s'engage à protéger votre vie privée et vos données personnelles. Cette politique de confidentialité explique de manière transparente comment nous collectons, utilisons, stockons et protégeons vos informations personnelles lorsque vous utilisez notre application de quiz éducatif disponible sur Google Play Store.

Engagement RGPD : Cette application respecte intégralement le Règlement Général sur la Protection des Données (RGPD) et les politiques de confidentialité de Google Play.

2. Informations que Nous Collectons

2.1 Données Collectées Automatiquement

  • Données de Performance : Scores de quiz, statistiques de jeu, temps de réponse, catégories préférées
  • Données Techniques : Version de l'application, système d'exploitation, modèle d'appareil, langue
  • Données d'Utilisation : Fréquence d'utilisation, fonctionnalités utilisées, durée des sessions
  • Identifiants Publicitaires : ID publicitaire Android (GAID) pour les publicités personnalisées via Google AdMob

2.2 Données que Vous Fournissez Volontairement

  • Préférences Utilisateur : Paramètres de jeu, thème choisi, options audio et timer
  • Scores et Classements : Vos résultats de quiz et records personnels
  • Nom d'Utilisateur : Pour les classements locaux (facultatif)

2.3 Données des Services Tiers

  • Google AdMob : Collecte des données pour afficher des publicités pertinentes
  • Google Play Services : Pour les achats in-app et services Google

3. Comment Nous Utilisons Vos Informations

3.1 Fonctionnalités Principales

  • Fournir les fonctionnalités de quiz et de jeu
  • Sauvegarder vos scores et statistiques
  • Personnaliser votre expérience utilisateur
  • Améliorer les performances et la stabilité de l'application
  • Fournir un support technique

3.2 Publicités et Monétisation

  • Afficher des publicités pertinentes via Google AdMob
  • Mesurer l'efficacité des campagnes publicitaires
  • Gérer les achats in-app pour la version premium

3.3 Amélioration du Service

  • Analyser les tendances d'utilisation (de manière anonymisée)
  • Développer de nouvelles fonctionnalités
  • Corriger les bugs et améliorer la performance

4. Base Légale du Traitement (RGPD)

Nous traitons vos données personnelles sur les bases légales suivantes :

  • Consentement : Pour les publicités personnalisées et les données facultatives
  • Exécution du contrat : Pour fournir les services de l'application
  • Intérêt légitime : Pour améliorer l'application et assurer sa sécurité
  • Obligation légale : Pour respecter les lois applicables

5. Stockage et Sécurité des Données

5.1 Stockage Local

Vos scores, préférences et statistiques sont stockés localement sur votre appareil. Ces données ne sont pas transmises à nos serveurs et restent sous votre contrôle.

5.2 Données des Services Tiers

Les données collectées par Google AdMob sont stockées selon les politiques de Google. Nous n'avons pas accès à ces données personnelles.

5.3 Mesures de Sécurité

  • Chiffrement des communications avec les services tiers
  • Stockage sécurisé local sur votre appareil
  • Accès limité aux données strictement nécessaires
  • Mise à jour régulière des mesures de sécurité

6. Partage des Données

Nous ne vendons jamais vos données personnelles.

Nous pouvons partager des données uniquement dans les cas suivants :

  • Services Essentiels : Google AdMob pour les publicités, Google Play pour les achats
  • Obligation Légale : Si requis par la loi ou une autorité compétente
  • Consentement Explicite : Avec votre autorisation préalable
  • Protection : Pour protéger nos droits, votre sécurité ou celle d'autrui

7. Vos Droits (RGPD)

Vous disposez des droits suivants concernant vos données personnelles :

7.1 Droits d'Accès et de Contrôle

  • Droit d'accès : Consulter vos données via les paramètres de l'application
  • Droit de rectification : Modifier vos préférences et informations
  • Droit d'effacement : Supprimer vos données via les paramètres
  • Droit à la portabilité : Exporter vos données (contactez-nous)

7.2 Droits de Limitation

  • Droit d'opposition : Vous opposer au traitement de vos données
  • Droit de limitation : Demander la limitation du traitement
  • Retrait du consentement : Retirer votre consentement à tout moment

7.3 Comment Exercer Vos Droits

  • Via les paramètres de l'application
  • En nous contactant à : quizz4u.contact@gmail.com
  • Via les paramètres de votre appareil pour les publicités

8. Publicités et Services Tiers

8.1 Google AdMob

  • Nous utilisons Google AdMob pour afficher des publicités
  • AdMob peut collecter des données selon sa propre politique
  • Vous pouvez désactiver la personnalisation dans les paramètres Android
  • Politique AdMob : https://policies.google.com/privacy

8.2 Contrôle des Publicités

  • Android : Paramètres → Google → Publicités → Désactiver la personnalisation
  • Application : Version premium sans publicités disponible
  • Opt-out : Lien de désabonnement dans les publicités

9. Protection des Mineurs

Notre application est conçue pour être éducative et familiale. Nous ne collectons pas sciemment d'informations personnelles d'enfants de moins de 13 ans sans le consentement parental. Si vous êtes parent et pensez que votre enfant nous a fourni des informations personnelles, contactez-nous immédiatement.

10. Transferts Internationaux de Données

Vos données peuvent être traitées dans des pays autres que votre pays de résidence, notamment aux États-Unis (serveurs Google). Ces transferts sont effectués avec des garanties appropriées conformément au RGPD.

11. Conservation des Données

  • Données locales : Conservées jusqu'à désinstallation de l'application
  • Données publicitaires : Selon les politiques de Google AdMob
  • Données de support : Supprimées après résolution du problème

12. Modifications de cette Politique

Nous pouvons mettre à jour cette politique de confidentialité pour refléter les changements dans nos pratiques ou pour des raisons légales. Les modifications importantes vous seront notifiées via l'application ou par email. Nous vous encourageons à consulter régulièrement cette politique.

13. Conformité et Certifications

Cette politique respecte :

  • ✅ Règlement Général sur la Protection des Données (RGPD)
  • ✅ Politiques de Google Play Store
  • ✅ Directives de Google AdMob
  • ✅ California Consumer Privacy Act (CCPA)
  • ✅ Lois sur la protection des données applicables
